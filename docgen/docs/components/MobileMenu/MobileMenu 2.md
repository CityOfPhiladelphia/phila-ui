# Basic Mobile Menu

A basic mobile menu layout intended to be used inside the 'menu-mobile' slot of [AppHeader](../AppHeader/) component.

E.g.

```
  <template v-slot:mobile-menu>
    <mobile-menu>
      <!-- Your menu here. -->
    </mobile-menu>
  </template>
```
